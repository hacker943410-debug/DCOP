# AI Working Agreement

## Defaults

- The main orchestrator owns task decomposition, subagent routing, integration, validation, and final reporting.
- Subagents may be used automatically when useful.
- Human approval is required for high-risk changes.
- Every task report includes files changed, validation, risks, and next action.

## High-Risk Changes

- Auth, payment, database, API contracts, deployment, secrets, and security policies require explicit approval.
