# Glossary

| Term | Meaning |
|---|---|
| Orchestrator | Main Codex session responsible for planning, delegation, integration, validation, and reporting |
| Subagent | Focused Codex role configured under `.codex/agents/` |
| Spark | `gpt-5.3-codex-spark`, used only for small bounded iterations |
| Spark dedup | Policy that disables duplicate GPT-5.5 code review for Spark-owned small diffs |
| ADR | Architecture Decision Record |
| PRD | Product Requirements Document |
